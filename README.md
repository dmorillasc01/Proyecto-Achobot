# Proyecto-Achobot
#### El proyecto Achobot trata sobre darle mas vida a nuestro amigo, con nuevas funcionalidades y conexiones automáticas en Linex y Windows.

Para ejecutar el programa, diríjase a la ruta Proyecto-Achobot --> dist --> y haga doble-click sobre "Achobot APP".

En el caso de que se haya descargado la versión de Windows (Recomendada si eres usuario de Windows), solo debes ejecutar el programa pinchando en "Achobot APP".
